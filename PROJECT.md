# オリジナルコーヒードリップバッグ ホームページ

## 概要
オリジナルコーヒードリップバッグブランドのホームページ。一般のお客様に向けて、商品の魅力・ブランドのこだわりを伝えながら、購買・SNSへの誘導を行うブランドサイト。

## ターゲットユーザー
一般のお客様（コーヒー好き、プレゼント探し中の方、ブランドに興味を持った方）

## 主要機能（v1）
- [x] ヒーローセクション（キャッチコピー + CTA ボタン）
- [ ] 商品紹介・ラインナップセクション（商品カード一覧）
- [ ] ブランドストーリー・こだわりセクション
- [ ] SNS（Instagram）連携セクション（インスタリンク + 最新投稿風グリッド）
- [ ] お問い合わせフォーム（Firestoreに保存）

## 技術設計

### ページ構成
- `/` — トップページ（Hero → 商品ラインナップ → ブランドストーリー → SNS → お問い合わせ）
  ※ 1ページのランディングページとして完結。セクションをスクロールで辿る構成。

### Firestore データ構造
contacts/{contactId}:
  - name: string         // お名前
  - email: string        // メールアドレス
  - message: string      // メッセージ本文
  - createdAt: timestamp
  - updatedAt: timestamp

### Firebase 利用サービス
- Firestore: お問い合わせフォームの送信データを保存
- Auth: 不要（管理者機能はv2以降）
- Storage: 不要（商品画像はpublicフォルダで管理）

### 商品データ（初期はハードコード）
商品情報は `src/data/products.ts` にハードコードで管理。
ECサイト連携が必要になった時点でFirestoreへ移行。

### 追加ライブラリ
- `react-intersection-observer` — スクロールアニメーション用（インストール済みでなければ追加）

### 実装順序
1. グローバルレイアウト（ナビゲーションヘッダー + フッター）
2. ヒーローセクション（キャッチコピー・画像・CTAボタン）
3. 商品ラインナップセクション（カードグリッド）
4. ブランドストーリー・こだわりセクション
5. Instagram連携セクション（リンク誘導 + グリッドレイアウト）
6. お問い合わせフォーム（react-hook-form + zod + Firestore保存）
